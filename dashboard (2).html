🎓 Learn Caesar Cipher

What is the Caesar Cipher?

A substitution cipher where each letter is shifted by a fixed number of positions in the alphabet. Named after Julius Caesar, who used it for military correspondence.

Key Terms

  • Plain Text — your original message
  • Cipher Text — the encrypted output
  • Shift / Key — number of positions to rotate

How It Works

With shift 3: A→D, B→E, C→F …

Advantages

  • Simple & fast
  • Great for teaching

Limitations

  • Only 25 possible keys
  • Vulnerable to frequency analysis

Timeline

  1. ~58 BC — Julius Caesar uses shift-3 for Gallic Wars dispatches.
  2. 9th century — Al-Kindi develops frequency analysis, breaking Caesar.
  3. Today — Used in education, ROT13 forums, and CTF challenges.

Real-World Applications

  • ROT13 (shift 13) in online forums for spoiler text
  • Introductory cryptography curricula
  • Capture-The-Flag (CTF) puzzles
