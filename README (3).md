# CipherShield Text Pro

Advanced Caesar Cipher Encryption & Decryption Platform — built as a premium cybersecurity SaaS workspace with **HTML5, CSS3, and Vanilla JavaScript only** (no frameworks, no backend).

## Files
- `login.html` — secure login page
- `dashboard.html` — full cipher workspace
- `style.css` — glassmorphism + neon design system
- `script.js` — cipher engine, particles, charts, storage

## Features
Encrypt / Decrypt · Step-by-step visualization · Alphabet mapping · Caesar Cipher Playground with rotating alphabet wheel · Brute-force demo · Frequency analysis · History (search / CSV import & export) · Analytics with animated counters · Learning mode · Settings (theme, accent, font, animation speed) · Toast notifications · Keyboard shortcuts · Fully responsive.

## Run
Open `/login.html`. Demo login accepts any username and a password ≥ 4 chars.
